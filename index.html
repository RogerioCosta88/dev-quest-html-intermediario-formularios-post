Formulários POST

O post é o outro valor possível para escolher no atributo method.

Diferente do get que passa as infomações como query string, o post envia essas informações de forma mais segura.

Possibilitando o envio de senhas e infomações pessoais.

Já o atributo action é usado para definir para onde as informações serão enviadas. Que pode ser para um arquivo interno, como, index.php ou outra seção do site como o Google faz.
